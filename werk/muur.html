Muur voor de energietransitie

Interactief bewust worden van energie en geld verspilling
Februari - Juni 2023
Terug

Om bewustwording van de energietransitie te vergroten onder jongeren heb ik met een projectgroep een interactieve muur ontworpen. We hebben energie besparen aan geld besparen gekoppeld omdat dit interesse kan opwekken passend bij de doelgroep. Ik heb hiervoor onderzocht hoeveel geld je jaarlijks bespaart bij bewuste keuzes in energieverbruik. Deze informatie heb ik op een speelse manier verwerkt op een interactieve muur met behulp van Makey Makey en code (HTML CSS).

Gebruikerstesten


Om de gebruiksvriendelijkheid van interactie met de muur, en de informatie daarbij, te testen hebben we aan mensen binnen de doelgroep om feedback gevraagd bij de ervaring. Hieruit kon ik samen met een teamgenoot pijnpunten vinden om te verbeteren in de uiteindelijke muur. Testen met een make-belief muur hielp met het voorkomen van negatieve ervaringen, en was makkelijk om mee te testen vroeg in het proces voor het maken van de werkende interactieve muur.

Makey makey + code


De interactieve muur werkt met het gebruik van een Makey Makey en HTML en CSS code. Ik was verantwoordelijk voor het samenstellen van het systeem voor de input (Makey Makey) en output (code) van de muur. De idee van Makey Makey is dat alles een knop kan zijn met stroomkringen, en deze knoppen kunnen verschillende toetsen op een laptop indrukken. Het indrukken van deze toetsen zogt er (door de HTML CSS code) voor dat verschillende video's afspelen als feedback op interactie met de muur. Dit was mijn eerste ervaring met het schrijven van code voor interactie en het werkte goed wanneer het koperdraad aangesloten was.
