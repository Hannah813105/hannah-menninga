Deltarune Character Index

Ervaring opdoen met React Js en JSON
Februari - April 2024
Terug

Tijdens mijn tweede studiejaar heb ik het keuzevak Front-end Development gevolgd. Voor dit vak was de opdracht om gebruik te maken van een code framework en javascript om informatie uit een API te visualiseren met interactie. Hiervoor heb ik samen met twee teamgenoten een eigen .JSON file gemaakt en samengewerkt binnen een github repository.

React


Voor het framework hadden we de keuze gemaakt om React.js te gebruiken, omdat het een flexibel en duidelijk framework is voor beginners vergeleken met Vue en Angular. Daarbij is er ook meer informatie te vinden over het gebruik van react met een actieve community online. We hebben een repository met React js opgezet en binnen de public folder met html, css en javascript gewerkt. Om elkaars werk te controleren en beter samen te werken voerde we in deze repository code reviews uit voordat deze gepusht werden naar de branch waarin we werkte. De volgende stap was een eigen library zoeken om te visualiseren met code.

.JSON


Voor de opdracht heb ik met mijn teamgenoten gekozen voor de game Deltarune als onderwerp. Voor dit onderwerp was er online geen API te vinden om mee te werken dus we kozen ervoor om zelf een .JSON file van Deltarune karakters samen te stellen. Om dit te doen hebben we Google Sheets gebruikt en per categorie informatie ingevuld om later aan te roepen in de React website. Ik was hiervoor verantwoordelijk voor de structuur waarin we werkten. Met alle informatie over de karakters verzameld besloten we een index/informatiepagina te maken voor deze karakters met een filtersysteem om ze in te zoeken.

Filter systeem + Grid


Voordat we bezig gingen met de code hebben we een design gemaakt voor het filtersysteem. Het concept van dit systeem was dat je met filters kon zoeken en dan een karakter te zien krijgt met informatie gerelateerd aan de game en een passende achtergrond. Ik was tijdens het coderen verantwoordelijk voor de UI-design dus ik hield me vooral bezig met het grid waarin de elementen zich bevonden. Ik heb ook in javascript mijn teamgenoot ondersteund voor de werking van de knoppen van de filters.
